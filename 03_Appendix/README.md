# Appendices

This folder contains reference materials: notation atlases, glossaries,
object maps, figure maps, and future technical appendices to the main
corpus.

The current main corpus is located in [`../00_Core`](../00_Core).
Appendices are not the entry point into the theory; they help compare
notation, construction statuses, and figures after reading the main
volumes.

## Current Content

- [`DOT_Appendix_AF_Atlas_And_Glossary_EN.md`](DOT_Appendix_AF_Atlas_And_Glossary_EN.md)  
  earlier English object atlas and notation glossary from the DOT
  package. It is kept as a reference source. In the current English DOT
  package it still needs revision according to the terminology and order
  of Volumes 0-6.

## Planned Appendices

Planned English appendices:

1. atlas of carriers, layers, relations, and operators;
2. glossary of statuses: standard construction, DOT reading, bridge,
   hypothesis, appendix;

4. catalogue of verified claims and corresponding scripts;
5. table relating old DOT notation to current DOT terminology.

Until these files are prepared, the main reference entries are the README
files in [`../00_Core`](../00_Core), [`../01_Verification`](../01_Verification),
and [`../02_Bridges`](../02_Bridges).
