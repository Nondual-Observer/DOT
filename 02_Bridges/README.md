# Bridges and Applications

This folder contains materials related to DOT that are not the entry point
into the main Volumes 0-6 corpus.

## Current Bridge Status

### 01_Color_RGB_Kuhn

Color projection of rank \(3\): RGB/CMY, chromatic octahedron, chambers,
and Kuhn sectors.

Status: preserved as a future bridge. The color line is outside the main
Volumes 0-6 corpus.

### 02_Binary_Growth

Earlier notes on binary growth.

Status: part of this content enters Volumes 0 and 2. The folder is kept as
a historical source.

### 03_A2_sl3_su3

Algebraic reading of the six-state structure through \(A_2\),
\(\mathfrak{sl}_3\), and \(\mathfrak{su}(3)\).

Status: bridge. The verification script has been moved to
`01_Verification`.

### 04_Hopf_Borromean

Topological reading of \(R_3=3K_2\), axial pairs, finite holonomy
\(T^3=\kappa\), Hopf pairs, and Borromean triad.

Status: the main content was revised into Volume 6. The folder is kept as
a source and extension.

### 05_Cryptographic_Spectral_Block

Spectral-cryptographic package for the composite graph.

Status: appendix. The verification script has been moved to
`01_Verification`.

### 06_Logic_Operator_Reading

Logic-operator reading and the operator tower.

Status: the main content was revised into Volume 4. The folder is kept as
a source.

### 07_Fano_Tetrahedral_Barycentric_Reading

Fano-tetrahedral and barycentric reading of rank \(4\).

Status: the main content was revised into Volume 5. The folder is kept as
a source.

### 08_Boundary_Operator_And_Distinction

Boundary operator, distinction, \(\partial,\delta\), and compatibility
with \(\kappa\).

Status: the main content was revised into Volume 3. The folder is kept as
a source.

## What Enters the Main Corpus

The main corpus already includes:

- from `04_Hopf_Borromean`: cyclic-topological reading of rank \(3\);
- from `06_Logic_Operator_Reading`: the operator tower;
- from `07_Fano_Tetrahedral_Barycentric_Reading`: Fano-tetrahedral
  architecture of rank \(4\);
- from `08_Boundary_Operator_And_Distinction`: boundary operators and
  scene reductions.

The remaining bridges stay outside the current main corpus.
