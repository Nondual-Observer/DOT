# ТНР: Полная развёртка `L1` без формальных доказательств

Этот документ даёт прямую развёртку текущего `L1` без формальных доказательств
и без дополнительного объяснительного текста. Здесь фиксируются только
несущая основа, путь сборки, `δ₀`, `Ω`, `T` и итоговая схема для всех `24`
частиц.

## Общий закон

```text
M = B
M = B · exp(δ₀ + Ω)
M = B · exp(δ₀ + Ω) · T
```

где:

- `B` — несущая основа;
- `δ₀ + Ω` — грамматика сдвигов;
- `T` — грамматика хвоста.

Ниже частицы развёрнуты по `5` суперпутям и `15` операторным путям ровно так,
как это сейчас реализовано в движке.
