# Readable Reports

Эта папка хранит generated outputs,
полученные напрямую из человекочитаемых report-скриптов.

Здесь лежат не исходники, а сохранённые reference-artifacts:

- `l1_particle_readable_report.txt` — полный каталог L1-частиц;
- `l4_element_readable_report.txt` — полный каталог 118 элементов;
- `l4_element_example_U92.txt` — развёрнутый пример для `U (Z=92)`.

Назначение этой папки:

- сохранять итоговые текстовые readout'ы рядом с кодом;
- делать структуру репозитория самодостаточной;
- не заставлять каждый раз перегенерировать базовые reference-отчёты вручную.
