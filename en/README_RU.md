## Что делает этот код

Этот репозиторий содержит **исполняемую численную верификацию** Теории Различимой Наблюдаемости (ТНР) — набор Python-скриптов, которые воспроизводят все предсказания теории **без единого свободного параметра**.

Все константы выводятся из геометрии одного объекта — **Октаэдра K(2,2,2)** (6 вершин, 12 рёбер, 8 граней).

### Быстрый запуск

```bash
cd ../companion_code
pip install -r requirements.txt    # только numpy

# Запустить всё:
python3 scripts/dot_companion_verify.py --all

# Или запустить отдельные модули:
python3 scripts/dot_companion_verify.py --full        # Дашборд 21 наблюдаемой
python3 scripts/dot_companion_verify.py --vacuum64    # Движок 64 состояний вакуума
python3 scripts/dot_companion_verify.py --chem        # Энергии молекулярных связей
python3 scripts/dot_companion_verify.py --tails       # Калькулятор хвостов Меллина
python3 scripts/dot_companion_verify.py --hydrogen-dyn  # Динамика на графе Кэли (48-dim)
```

### Что проверяет каждый модуль

| Модуль | Что проверяет | Ключевая метрика |
|:-------|:--------------|:-----------------|
| `--full` | 21 наблюдаемая: 9 масс частиц, 4 константы связи, 4 космологических фракции, g-2, тонкая структура, логарифм Бете | Средняя ошибка: **0.0002%** |
| `--vacuum64` | 64 состояния вакуума K(2,2,2), спектральные проекции → заряды кварков | Все 1/3, 2/3, 1/4 — **точно** |
| `--chem` | 14 молекул (H₂, H₂O, CH₄, NH₃, HF, C₂H₂, C₂H₄, C₂H₆ и др.) | Средняя ошибка: **0.32%** |
| `--tails` | Полный атлас масс 26 частиц с поправками Меллина | Построчный разбор δ₀ + Ω |
| `--hydrogen-dyn` | Динамика распада 2p→1s на 48-мерном графе Кэли | P(1s) → 98.7%, норма = 1.000 |

## Обзор теории

**Теория Наблюдаемых Различий (ТНР)** предлагает рассматривать всю архитектуру фундаментальной физики как производную от комбинаторных и спектральных свойств одного графа: **полного трипартийного графа K(2,2,2)** — 1-скелета правильного октаэдра.

Из этой единственной структуры, **без свободных параметров**, ТНР выводит:

- **Постоянную тонкой структуры**: α⁻¹ = 432/π − 47γ³/2 ≈ 137.036 (ошибка < 0.001%)
- **Массы частиц**: 26 частиц по единой формуле M = C/γᵏ · mₑ · exp(δ₀ + Ω)
- **Космологические фракции**: Ωb = 4/81, Ωc = 21/81, ΩΛ = 56/81 (точное замыкание Σ = 1)
- **Кварковые заряды**: 1/3, 2/3 вычисляются как спектральные проекции K(2,2,2)
- **Угол Вайнберга**: sin²θ_W = 2/9 = 3γ²
- **Энергии молекулярных связей**: H₂, CH₄, H₂O, HF, … со средней точностью 0.3%

где γ = √6/9 — **анизотропия наблюдателя**, уникальный геометрический инвариант октаэдра.

## Три принципа: архитектура оператора

Вся теория строится на **трёх взаимосвязанных принципах**, каждый из которых вытекает из предыдущего:

### 1. Закон замкнутости: ∂² = 0

В природе нет ничего, что существовало бы само по себе. Любое напряжение обязано найти свою абсолютную компенсацию — все волны и асимметрии должны в сумме давать ноль.

Это **Принцип Замкнутости** (∂² = 0): граница границы равна нулю. Ни одна структура не может существовать, если она не замыкается сама на себя.

### 2. Парадокс грани: тройка не делится на двойку

Минимальная объёмная структура, совместимая с ∂² = 0, — это **Октаэдр** (3 оси × 2 полюса = 6 вершин). Но грань октаэдра — это **треугольник** (3 ребра), а базовая логика информации — **бинарная** (да/нет, 0/1 — 2 состояния).

Возникает фундаментальный парадокс: стенка из 3-х линий пытается симметрично «вжаться» в логику 2-х. Это невозможно без остатка. **Тройка не делится на двойку.**

Этот крошечный «заусенец» — топологическая ошибка, которая не исчезает. Она и есть источник всей физики: из неё рождается анизотропия наблюдателя γ = √6/9.

### 3. Закон непроницаемости: магия взаимно простых чисел

Чтобы «размазать» эту ошибку, вакуум вынужден строить вокруг неё защитные слои — горизонты, вложенные как матрёшка. Каждый слой состоит из **Внутреннего Тела** (например, число 80) и **Внешнего Фона** (число 81).

Но почему эти слои не слипаются обратно в ноль? Потому что два соседних числа всегда **взаимно просты**: у 80 и 81 нет ни одного общего простого делителя. Их внутренние частоты абсолютно ортогональны.

Тело пульсирует на своих частотах (двойки, пятёрки), а Фон — на своих (тройки). Звук Тела бьётся об стенку Фона и **стопроцентно отражается обратно**, не имея возможности раствориться. Стенка непробиваема.

> **Итог:** ∂² = 0 порождает октаэдр → несовместимость 3 и 2 порождает γ → взаимная простота запирает структуру в стабильные горизонты. Из этих трёх слоёв вырастает **всё**: массы частиц, космологические фракции, заряды кварков.
