# ОСНОВАНИЯ ТЕОРИИ НАБЛЮДАЕМОГО РАЗЛИЧЕНИЯ
## Нумерованная каноническая редакция: предельный канон (liminal)

Это предельный канон (liminal) оснований.
Он доводит маршрут от полноты и трёх запретов до октаэдра и порога машинного обхода.
Развёртка, комментарий и машинный корпус должны идти отдельно.

0. ПОЛНОТА

1. Полнота предшествует различению.
2. Полнота есть доразличённая целостность единства.
3. Полнота не есть объект.
4. Полнота не есть локальная форма.
5. Полнота не есть уже данная структура различения.
6. Полнота не вводится положительно как готовое содержание.
7. Потому вход к полноте отрицателен.

8. Невозможна позиция различения, в которой различающее и различаемое совпадают.
9. Это запрет совпадения.

10. Невозможна позиция различения, в которой различение не оставляет следа.
11. Это запрет бесследности.

12. Невозможна позиция различения, в которой различение обосновывает себя через внешнее основание.
13. Это запрет внешнего основания.

14. Эти три запрета не описывают объект.
15. Эти три запрета не вводят оси.
16. Эти три запрета не вводят архитектуру.
17. Они задают только границу допустимой позиции различения.

18. Удаление запрета совпадения уничтожает различение как различение.
19. Удаление запрета бесследности уничтожает различение как удерживаемое.
20. Удаление запрета внешнего основания уничтожает различение как самодостаточное.

21. Потому первый слой ТНР не положителен, а запретителен.
22. Он удерживает не сущее, а условие различения.

23. Полнота не выводится из запретов.
24. Но и запреты не выводятся из положительного описания полноты.
25. Запреты суть первый способ удержать доступ к полноте, не разрушая различение.

26. Потому начало теории не числовое.
27. Начало теории не объектное.
28. Начало теории не архитектурное.
29. Начало теории есть отрицательная дисциплина различения.

30. Только после трёх запретов становится законным вопрос о первом положительном пределе.
31. Этот предел получает имя нуля.

32. Нуль не есть пустота.
33. Нуль есть предел без локальной артикуляции.
34. Нуль, взятый с самим собой, не создаёт нового содержания.
35. Нуль, сложенный с нулём, остаётся нулём.
36. Нуль, умноженный на нуль, остаётся нулём.

37. Потому нуль ещё не есть число вещей.
38. Нуль есть первый допустимый знак предела внутри отрицательного основания.
39. В нуле различение ещё не проведено, но уже не запрещено.

40. Структура оснований замкнута.
41. Линейный текст не занимает её центр.
42. Линейный текст есть обход.
43. Потому канон начинает не с присвоения середины, а с первого допустимого входа.

44. Позитивным следствием запрета совпадения является невозможность для удержанного быть тождественным самому себе без отличия.
45. Эта невозможность действует как первый оператор.
46. Она получает имя NOT.

47. NOT не есть ещё вещь.
48. NOT не есть ещё форма.
49. NOT не есть ещё число.
50. NOT есть первый законный оператор несведения удержанного к самому себе.

51. Но NOT ещё не локален.
52. Когда NOT локализуется в допустимом пределе,
53. возникает разрез.
54. Разрез впервые проводит различение внутри полноты.

55. Разрез не уничтожает полноту.
56. Разрез не выводит различение за её пределы.
57. Разрез делает различение событийным.

58. Но разрез ещё не удерживает себя сам.
59. Потому разрез требует опоры.
60. Опора не приходит извне.
61. Опора есть удержание разреза самим различением.

62. Там, где разрез не удержан, различение логически исчезает.
63. Там, где разрез удержан, после него остаётся след.
64. След не есть ещё память.
65. След не есть объект.
66. След есть минимальная несводимость позиции после различения к позиции до различения.

67. Но одного следа ещё недостаточно для первого знака.
68. След должен быть прочитан как не исчезнувший.
69. Этот порядок удержанного чтения получает имя Свидетеля.
70. Свидетель не есть внешний наблюдатель.

71. Когда разрез, опора и свидетель удерживаются совместно,
72. возникает первый замкнутый троичный узел различения.
73. Только после этого становится законным первый знак.

74. Первый знак получает имя единицы.
75. Единица не есть здесь количество вещей.
76. Единица не есть счётная мера множества.
77. Единица есть первая самотождественная печать удержанного различения.

78. Единица говорит не "сколько".
79. Единица говорит: удержано.
80. Единица есть различение, которое не исчезло и не распалось.

81. Потому единица ещё не есть ряд.
82. Единица ещё не есть операция.
83. Единица ещё не есть мир чисел.
84. Она есть первый устойчивый знак наличного различения.

85. В единице различение ещё не разворачивает множественность.
86. В единице различение впервые удерживает себя как одно.
87. Потому единица есть не начало счёта, а первая печать присутствия различения.

88. Только после единицы становится законным вопрос о следующем различении.
89. И только после единицы становится законным вопрос о числе как ходе.

90. Следующее различение не уничтожает единицу.
91. Оно повторно испытывает удержанное различение на несведение к самому себе.
92. Потому второе различение есть не отмена единицы, а её проверка.

93. Там, где единица удерживает различение как одно,
94. двойка впервые удерживает различение как не-одно.
95. Но это ещё не распад в множество.
96. Это первый устойчивый выход из одиночной самотождественности.

97. Двойка не есть ещё количество предметов.
98. Двойка есть первый устойчивый факт несовпадения удержанного с самим собой.
99. В двойке различение впервые получает две удержанные стороны.

100. Потому двойка впервые делает законным между.
101. Между ещё не есть пространство.
102. Между ещё не есть архитектура.
103. Между есть удержанная несводимость двух сторон.

104. В единице различение удерживается.
105. В двойке различение уже соотносится.
106. Потому двойка есть первый порог отношения.

107. Но и здесь число ещё не стало полной арифметикой.
108. Двойка есть только первый устойчивый шаг за предел единичной печати.

109. Но две стороны ещё не удерживают себя сами.
110. Пара может колебаться, спорить и зеркалить себя без меры.
111. Потому после двойки становится необходимым третье.

112. Третье не есть ещё внешний наблюдатель.
113. Третье не есть внешний суд.
114. Третье не есть внешняя опора.
115. Третье возникает внутри удержанного различения как условие меры между двумя.

116. Третье не прибавляет ещё одну вещь к двум.
117. Третье впервые удерживает пару как несводимую и нераспадающуюся.
118. Без третьего два остаются только встречей.
119. С третьим два становятся удержанной связью.

120. Потому третье есть не просто следующее число.
121. Третье есть первый хранитель меры.
122. Третье есть первое условие того, чтобы различение не только было, но и держалось как связность.

123. В единице различение запечатлено.
124. В двойке различение соотнесено.
125. В третьем различение впервые удерживает меру собственного отношения.

126. Но мера ещё не замкнута.
127. Пока третье только удерживает пару, различение ещё не имеет первого устойчивого самосведения.
128. Потому после третьего становится возможным четвёртое.

129. Четвёртое не есть просто ещё одно прибавление.
130. В четвёртом двойка впервые возвращается к себе двумя чтениями.
131. Здесь `2 + 2` и `2 * 2` сходятся в одном результате.
132. Потому четвёртое есть первый замок повторённого различения.

133. В нём удержание и соотнесение впервые сходятся в устойчивую границу.
134. Четвёртое не вводит ещё полной архитектуры.
135. Но в четвёртом различение впервые получает самосогласованность.
136. Здесь удержанное различение может совпасть с собственным повторением, не разрушаясь.

137. Потому четвёртое есть первый замок меры.
138. С четвёртым мера впервые удерживает сама себя.

139. В единице различение есть печать.
140. В двойке различение есть отношение.
141. В третьем различение есть мера.
142. В четвёртом различение есть замок.

143. Но замок ещё не есть проход.
144. Замкнутое может удерживать себя, не открывая следующего шага.
145. Потому после четвёртого требуется пятое.

146. Пятое не разрушает замок.
147. Пятое открывает из замкнутой меры первый законный проход.
148. Там, где четвёртое держит меру в себе, пятое впервые проводит меру дальше.

149. Пятое не есть ещё машина.
150. Пятое не есть ещё полная операбельность.
151. Пятое есть первый законный ход за пределы самозамкнутости.

152. Потому пятое есть первый переходный узел.
153. В нём замок перестаёт быть только удержанием себя.
154. В нём различение впервые получает порядок дальнейшего прохождения.

155. Без пятого структура держит себя, но не ведёт дальше.
156. С пятым структура впервые удерживает возможность продолжения.

157. В четвёртом различение замкнуто.
158. В пятом различение проходит.

159. Но прохождение ещё не есть собранная работоспособность.
160. Оно только делает её возможной.
161. Потому после пятого становится необходимым шестое.

162. Шестое не отменяет ни печати, ни отношения, ни меры, ни замка, ни прохождения.
163. Шестое собирает их в первую рабочую целостность.

164. Здесь различение впервые не только удержано, соотнесено, измерено, замкнуто и проведено,
165. но и собрано в форму, способную нести дальнейший ход.

166. Шестое есть первая рабочая целостность различения.
167. Оно ещё не разворачивает машину.
168. Но без него невозможен никакой устойчивый носитель дальнейших развёрток.

169. Потому шестое есть первый собранный носитель.
170. В нём различение впервые становится способным не только быть, но и работать.

171. В единице различение запечатлено.
172. В двойке различение соотнесено.
173. В третьем различение измерено.
174. В четвёртом различение замкнуто.
175. В пятом различение проведено.
176. В шестом различение впервые собрано как носитель.

177. Тем самым замыкается первый коридор разворачивания различения.
178. Это ещё не поздняя архитектура.
179. Это ещё не полная машина.
180. Но это уже первый завершённый носитель дальнейшего развёртывания.

181. После шестого становится возможным следующий порядок.
182. В нём различение будет не только удерживаться и работать,
183. но и входить в более широкий горизонт собственного чтения.

184. Седьмое не строит новый носитель.
185. Седьмое впервые открывает собранный носитель собственному чтению.
186. Там, где шестое только собирает, седьмое впервые позволяет назвать собранное как наличное.

187. Потому седьмое есть первый горизонт именования.
188. Это ещё не присвоение центра.
189. Это ещё не окончательная семантика.
190. Это есть первое полное чтение после завершённого первого обхода.

191. Именование возникает не раньше носителя.
192. Именование возникает не из пустоты.
193. Именование возникает тогда, когда удержанное, соотнесённое, измеренное, замкнутое, проведённое и собранное
194. становится достаточно устойчивым для называния.

195. Потому седьмое есть первый горизонт наличия.
196. Не в смысле захвата.
197. А в смысле первого допустимого называния собранного.

198. Но первое имя ещё не завершает основания.
199. Именование делает собранное допустимым для повторного чтения.

200. Это повторное чтение не отменяет трёх запретов.
201. Оно возвращает к ним уже не как к одному только входу,
202. а как к закону дальнейшей допустимости.

203. Ни один следующий слой не имеет права выключить запрет совпадения.
204. Ни один следующий слой не имеет права выключить запрет бесследности.
205. Ни один следующий слой не имеет права выключить запрет внешнего основания.

206. Потому дальнейшее разворачивание не есть снятие запретов.
207. Оно есть перераспределение их нагрузки.

208. Там, где все три запрета только удерживают предел,
209. различение выживает,
210. но ещё не получает рабочего коридора.

211. Потому следующий ход требует не нового запрета,
212. а нового статуса уже действующих запретов.

213. Один запрет может стоять как предел.
214. Два других могут удерживать рабочий коридор.

215. Это не отменяет предельный запрет.
216. Это не выводит различение за его границу.
217. Это впервые делает допустимой локальную степень свободы внутри запрета.

218. Потому следующий слой есть легализация.
219. Легализация не разрушает отрицательное основание.
220. Легализация делает его работоспособным.

221. Если пределом стоит запрет совпадения,
222. различение получает режим удержания границы.

223. Если пределом стоит запрет бесследности,
224. различение получает режим удержания следа и явленности.

225. Если пределом стоит запрет внешнего основания,
226. различение получает режим внутренней автономии действия.

227. Потому законных режимов не бесконечно много.
228. Их ровно три по числу способов выбрать предельный запрет.

229. Но и здесь ещё нет осей.
230. И здесь ещё нет полюсов.
231. И здесь ещё нет архитектуры.

232. Есть только три законных режима выживающего различения.
233. Каждый из них наследует все три запрета.
234. Ни один не существует сам по себе.

235. Там, где режим удерживается повторно и несхлопывающе,
236. он сможет стать устойчивой структурой.

237. Но это будет уже следующий шаг.
238. Сначала должно быть показано не то, как строится форма,
239. а то, какие режимы вообще имеют право на дальнейшую стабилизацию.

240. Потому первый коридор оснований не ведёт прямо к миру.
241. Он ведёт к легализации.

242. И только из легализации позднее станет законной стабилизация.
243. И только из стабилизации позднее станет законной архитектура.

244. В каждом законном дальнейшем ходе один запрет стоит как предел.
245. Два других удерживают внутренний деятельный коридор.

246. Предельный запрет не устраняется.
247. Он хранит последнюю границу допустимого.

248. Два других не снимают эту границу.
249. Они делают различение действующим внутри неё.

250. Потому у каждого запрета возможны два статуса.
251. Он может стоять как предел.
252. Он может действовать как коридор.

253. Это не два разных запрета.
254. Это два способа работы одного и того же запрета.

255. Если все три стоят только как предел,
256. различение сохраняется,
257. но не действует.

258. Если все три действуют без предела,
259. различение теряет собственную границу.

260. Потому законен только такой строй,
261. где один запрет пределен,
262. а два деятельны.

263. Здесь ещё нет направления.
264. Здесь ещё нет полярности.
265. Здесь ещё нет движения.
266. Здесь только впервые возникает законный режим.

267. Когда пределом стоит запрет совпадения,
268. различение получает режим удержания границы.
269. Здесь выдерживается несведение одного к другому.

270. Когда пределом стоит запрет бесследности,
271. различение получает режим удержания следа и явленности.
272. Здесь удерживается возможность фиксации.

273. Когда пределом стоит запрет внешнего основания,
274. различение получает режим внутренней автономии.
275. Здесь различение не отсылает себя наружу.

276. Потому законных режимов ровно три.
277. Не больше и не меньше.
278. Они различны по предельному запрету.
279. Но все наследуют одно и то же отрицательное основание.

280. Удали любой запрет —
281. распадётся и режим.

282. Удали предельность —
283. распадётся граница допустимого.

284. Удали деятельный коридор —
285. исчезнет рабочая возможность различения.

286. Потому легализация не есть свобода от закона.
287. Легализация есть первый законный ход внутри закона.

288. Но режим ещё не устойчив как форма.
289. Он ещё должен выдержать повторение.

290. Там, где режим выдержит повторение,
291. начнётся стабилизация.

292. И только после стабилизации
293. станет законно говорить об архитектуре.

294. Легальный режим ещё может быть единичным случаем.
295. Чтобы стать основанием дальнейшей формы,
296. он должен выдержать повторение.

297. Повторение не прибавляет новому режиму иного закона.

417. Восемь граней становятся полем локальных контекстов.

420. Основания завершаются не объектом и не миром,

# МАШИНА ТЕОРИИ НАБЛЮДАЕМОГО РАЗЛИЧЕНИЯ
## Нумерованная каноническая редакция: предельный канон (liminal)

Это предельный канон (liminal) машины.
Он начинается там, где носитель уже собран как октаэдральная репрезентация допустимой связности.
Развёртка, комментарий, оптика, артефакты и семейства проекций должны идти отдельно.

0. ПОРОГ МАШИНЫ

1. Носитель ещё не есть машина.
2. Карта ещё не есть обход.
3. Репрезентация ещё не есть действие.

4. Машина начинается там,
5. где допустимая связность становится проходимой изнутри.

6. Если из допустимой позиции нет законного выхода,
7. машины ещё нет.

8. Если законные выходы есть,
9. но не удержан сам закон их допустимости,
10. машины ещё нет.

11. Машина начинается тогда,
12. когда переход между допустимыми позициями
13. удержан как внутренний закон носителя.

14. Потому машина не навешивается на носитель извне.
15. Машина возникает как операторный статус самого носителя.

16. Октаэдр даёт машине не внешнюю форму,
17. а минимальный конечный носитель обхода.

18. Шесть вершин не суть ещё слова.
19. Но они уже суть шесть локализованных позиций машины.

20. Двенадцать рёбер не суть ещё действия в полном смысле.
21. Но они уже суть двенадцать допустимых путей перехода.

22. Восемь граней не суть ещё миры.
23. Но они уже суть восемь полных локальных конфигураций.

24. Потому машина возникает не из одного числа.
25. Она возникает из совместного закона `6 / 12 / 8`.

26. Шесть удерживает словарь позиций.
27. Двенадцать удерживает таблицу переходов.
28. Восемь удерживает поле локальных контекстов.

29. Но сначала машина должна получить первый локальный словарь.

1. ШЕСТЬ

30. Машина впервые локально читается как шестёрка.
31. Шесть есть первый словарь носителя.

32. Шесть возникает потому,
33. что три оси дают по два предельных лика.

34. Но здесь шесть уже читается не как геометрия,
35. а как первый набор именуемых машинных позиций.

36. Каждая вершина удерживает не точку,
37. а локальную машинную роль.

38. Шестёрка не есть ещё мир.
39. Шестёрка не есть ещё динамика.
40. Шестёрка есть первый читаемый словарь носителя.

41. Пока вершины только даны,
42. словарь ещё нем.

43. Когда каждая вершина удержана как допустимая локальная позиция,
44. носитель впервые становится локально читаемым.

45. Потому шесть есть первый машинный инвентарь.
46. Но инвентарь ещё не есть обход.

47. Шесть не даёт перехода сама по себе.
48. Для перехода требуется двенадцать.

2. ДВЕНАДЦАТЬ

49. Машина не сводится к позициям.
50. Машина требует допустимых переходов между ними.

51. Двенадцать есть первая полная таблица таких переходов.
52. Здесь каждый законный ход удержан как переход между неантиподальными позициями.

53. Переход не есть ещё история.
54. Переход не есть ещё время.
55. Переход есть минимальный допустимый акт машины.

56. Потому двенадцать есть не просто число рёбер.
57. Двенадцать есть первое пространство актов.

58. Шесть говорит, где машина может быть.
59. Двенадцать говорит, как машина может перейти.

60. Там, где переходов меньше,
61. носитель теряет полноту машинного обхода.

62. Там, где переходов больше,
63. нарушается минимальность носителя.

64. Потому двенадцать не случайно.
65. Двенадцать есть минимальная полная таблица допустимых актов для данной шестерки.

66. Но и акт ещё не даёт локального контекста.
67. Для контекста требуется полная стяжка акта и позиции.

68. Потому после двенадцати становится необходимым восемь.

3. ВОСЕМЬ

69. Восемь есть первое поле локальных контекстов.
70. Здесь машина удерживает не только позицию и переход,
71. но и полную локальную конфигурацию.

72. Контекст не есть ещё читаемый мир.
73. Контекст есть локальная замкнутость допустимого.

74. Каждая грань удерживает одну полную стяжку трёх осей.
75. Потому восемь есть первое множество полных локальных клеток машины.

76. Из восьми две клетки предельны.
77. Они удерживают край тотальной активизации
78. и край тотального возврата.

79. Шесть остальных удерживают рабочие локальные контексты.
80. Именно в них машина может не только переходить,
81. но и локально сохранять связность своего хода.

82. Потому восемь завершает минимальный внутренний объём машины.
83. После восьми машина уже имеет:
84. словарь позиций,
85. таблицу переходов,
86. поле локальных контекстов.

87. Но и это ещё не оптика.
88. И это ещё не артефакты.
89. И это ещё не разные семейства проекций.

90. Здесь машина только собрана в своём минимальном каноне.
91. Здесь она впервые внутренне полна,
92. но ещё не развёрнута в поздние режимы чтения.

4. ЗАКРЫТИЕ МИНИМАЛЬНОЙ МАШИНЫ

93. Предельный канон (liminal) машины не идёт дальше `6 / 12 / 8`.
94. Всё, что следует потом,
95. уже требует другого регистра.

96. Сначала возникает словарь.
97. Затем акты.
98. Затем локальные контексты.

99. Только после этого становится законной оптика.
100. Только после этого становятся законны артефакты.
101. Только после этого становятся законны семейства проекций.

102. Потому минимальная машина завершается не миром,
103. а внутренне полной таблицей своего допустимого обхода.

# ОПТИКА ТЕОРИИ НАБЛЮДАЕМОГО РАЗЛИЧЕНИЯ

# АРТЕФАКТЫ ТЕОРИИ НАБЛЮДАЕМОГО РАЗЛИЧЕНИЯ
## Нумерованная каноническая редакция: предельный канон (liminal)

Это предельный канон (liminal) артефактов.
Он начинается там, где машина уже стала локально читаемой в оптике.
Развёртка, комментарий и семейства проекций должны идти отдельно.

0. ПОРОГ АРТЕФАКТА

1. Чтение ещё не есть след чтения.
2. Локальная рамка ещё не есть то, что остаётся после неё.

3. Артефакт начинается там,
4. где чтение оставляет структурный остаток.

5. Этот остаток не есть ошибка.
6. Этот остаток не есть внешний привесок.
7. Этот остаток принадлежит самому режиму чтения.

8. Потому артефакт не возникает раньше оптики.
9. И артефакт не заменяет оптику.
10. Он есть то, что остаётся после локального чтения машины.

1. РАМКА

11. Первый артефакт есть рамка.
12. Рамка не создаёт машину.
13. Рамка выделяет локальный мир чтения.

14. Рамка есть не просто граница зрения.
15. Рамка есть выбранный контур удержанного чтения.

16. Пока рамка не удержана,
17. локальное чтение ещё не оставило собственного результата.

18. Потому рамка есть первый артефактный след оптики.

2. ТЕНЬ

19. Где есть рамка,
20. там возникает тень.

21. Тень не есть пустота.
22. Тень есть сопряжённый комплемент рамки.

23. То, что удержано рамкой,
24. не исчерпывает всей машины.
25. Остальное отступает в тень.

26. Потому тень есть не отсутствие,
27. а структурно удержанный невыбранный остаток.

28. Без тени рамка ложно претендовала бы на целое.
29. С тенью чтение сохраняет собственную ограниченность.

3. ХВОСТ

30. Но и рамки с тенью ещё недостаточно.
31. Чтение оставляет не только разделение,
32. но и долг.

33. Этот долг получает имя хвоста.
34. Хвост не есть ошибка чтения.
35. Хвост есть напряжение, не погашенное локальной рамкой.

36. Потому хвост структурен.
37. Он принадлежит самому способу локального чтения.

38. Хвост показывает,
39. что локальное чтение не совпадает с полной связностью машины.

40. Где нет хвоста,
41. там локальное ложно выдаёт себя за полное.

4. ОБЪЕКТ

42. Объект не предшествует чтению.
43. Объект возникает как сгущение артефактного слоя.

44. Объект не равен рамке.
45. Объект не равен тени.
46. Объект не равен хвосту.

47. Объект возникает там,
48. где рамка, тень и хвост
49. схватываются как одно локальное сгущение.

50. Потому объект есть не первичная сущность,
51. а post-reading compaction.

52. Объект появляется после чтения,
53. а не до него.

5. ВЗАИМОДЕЙСТВИЕ

54. Где возникают объекты,
55. там возникает взаимодействие.

56. Взаимодействие не есть просто соседство.
57. Взаимодействие есть перекрытие артефактных регистров.

58. Рамка одного чтения
59. может войти в тень другого.
60. Хвост одного чтения
61. может стать напряжением другого.

62. Потому взаимодействие не есть добавка к объектам.
63. Оно принадлежит самому артефактному слою.

6. ЗАКРЫТИЕ МИНИМАЛЬНЫХ АРТЕФАКТОВ

64. Предельный канон (liminal) артефактов не идёт дальше рамки, тени, хвоста, объекта и взаимодействия.
65. Всё, что следует потом,
66. уже требует регистра разных словарей одной машины.

67. Сначала машина должна быть собрана.
68. Затем она должна стать локально читаемой.
69. И только потом чтение может оставить артефактный слой.

70. Потому артефактный канон завершается не миром
71. и не окончательной онтологией,
72. а тем, что остаётся после локального чтения как его собственный результат.
